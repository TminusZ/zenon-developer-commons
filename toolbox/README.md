## Toolbox (Optional Builder Utilities)

This folder contains optional tools, scripts, and reference material for working with the protocol.

Nothing in this directory is required for:

* running a node
* implementing the spec
* interacting with the network

These are simply utilities that may help with:

* testing assumptions
* analyzing behavior
* exploring edge cases
* validating implementations

Use, modify, or ignore entirely.

The protocol does not depend on anything in this folder.
